Question-1: What are the different ways to select an element in the DOM?

We can select an element in the DOM by using the id selector document.getElementById(), class selector document.getElementsByClassName(), tag selector document.getElementByTagName() and by using the css selector document.querySelector() or querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

By innerHTML we can get and set the html markup with tags, attributes and text. By innerText we can get and set only the visible text of the html element. By textContent we can get and set all type of content of an html element. It contains hidden elements as well.
